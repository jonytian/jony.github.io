top命令

  • 1.top命令经常用来监控Linux的系统状况，比如cpu、内存的使用。

  • 2.第一行：

  • 3.第二行：

  • 4.第三行：cpu状态

  • 5.第四行：内存状态（单位k）

  • 6.第五行：swap交换分区

    可用内存数的近似的计算公式：第四行的free + 第四行的buffers + 第五行的cached。
    我们最关心的内存监控是第五行swap（类似于虚拟内存）交换分区的used，如果这个数值在不断的变化，说明内核在不断进行内存和swap的数据交换，这是可以判定服务器的内存不够用了。

  • 7.第六行：各进程（任务）的状态监控
